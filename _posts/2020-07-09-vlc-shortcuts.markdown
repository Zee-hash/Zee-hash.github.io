## Most used

| Key | Effect |  
| :---: | :---: |  
| `F` | Fullscreen |  
|`Esc` | Exit fullscreen mode |  
| `[/]` | Decrease/ increase playback by 0.1x |  
| `Space` | Play/Pause |
